# MonoList v0.4.3 QA 记录

> 测试日期：2026-07-06

## 本轮修复

- 菜单栏状态项恢复为原生 `NSStatusItem` 文字，不再设置自定义无障碍标题。
- 主清单使用无标题栏面板，去除透明标题栏造成的视觉空隙。
- 菜单栏锚点和 Dock 回退锚点均直接对齐菜单栏下边缘。
- 窗口背景不再接受拖动，只在表头预留的空白区域执行窗口拖动。
- 面板会在鼠标点击当前文本编辑器以外的位置时结束编辑，由现有失焦逻辑完成保存。
- 设置下拉框改为原生 `NSPopUpButton`，整个控件（包括箭头）均可触发。
- 更新按钮高度固定为 22 pt，更新状态和升级入口位于版本号右侧。

## 环境说明

当前受限测试环境无法访问图形会话中的 `NSScreen`，因此轻提醒面板的实机显示用例无法在本轮自动化进程内执行；提醒调度的非界面逻辑没有改动。本轮变更相关的构建和自动化检查均已通过。
