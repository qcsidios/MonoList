# MonoList v0.4.1 交互与发布设计

## 目标

本次迭代修复 v0.4.0 本地版本中的输入、窗口动画、间距、按钮和提醒问题，并直接发布为 v0.4.1。版本必须高于当前已安装的 v0.4.0，确保用户可以通过应用内更新完成升级。

## 一、Dock、菜单栏与窗口移动

- 点击 Dock 图标时，如果清单隐藏则显示清单；如果清单已经显示则将其置前，不执行关闭。
- 点击菜单栏“待办”时继续使用切换逻辑：第一次显示，再次点击隐藏。
- 清单默认紧贴菜单栏弹出。
- 清单显示后允许用户在屏幕可见范围内自由拖动。
- 窗口动态改变高度时，以当前窗口顶部为锚点，不改变用户拖动后的位置。
- 关闭后再次打开时恢复默认菜单栏下方位置。

## 二、新增与编辑

### 2.1 新增待办

- 点击加号后，在未完成任务末尾显示输入行并自动聚焦。
- 输入内容后按 Enter，保存当前待办，并立即在其下一行创建新的空输入行。
- 新输入行继续自动聚焦，支持连续录入多条待办。
- 空输入行按 Enter 不创建任务，也不继续增加空行。
- 输入内容后点击空白处，保存当前待办并结束连续输入。
- 空输入行点击空白处时取消，不创建任务。

### 2.2 编辑已有待办

- 双击已有待办进入编辑。
- 按 Enter 或点击任意空白位置均保存修改。
- 编辑结果为空时恢复修改前的内容，不删除任务。

## 三、窗口动画和间距

- 窗口外框尺寸只由 `WindowCoordinator` 控制。
- 历史记录显示、隐藏，以及任务新增、删除、完成时，窗口下边缘使用可见的平滑动画展开或收起。
- 动画过程中窗口顶部坐标保持不变。
- 连续高度请求只执行最后一个目标，避免动画互相覆盖。
- 列表内容固定从顶部开始排列，不允许内容因为剩余空间而垂直居中。
- 表头分割线、第一条未完成任务、已完成区域和窗口底边使用固定间距。
- “显示”和“隐藏”按钮使用相同固定宽度，文字切换时位置不变。

## 四、清单顶部按钮

- 加号、更多操作和设置均为 30 × 30 点圆角矩形。
- 三个按钮使用相同的背景、边框、悬停和按压反馈。
- 更多操作按钮不得被菜单控件压缩成小椭圆。
- 点击区域与可见背景范围一致。

## 五、控制台

- 顶部应用名称只显示“MonoList”，取消中文名称“一栏”。
- 其他中文界面文案保持不变。
- 提醒间隔和提醒位置控件由左侧文字区域及右侧正方形下拉按钮组成。
- 两个下拉控件的总宽度与“立即测试”按钮一致。
- 下拉按钮使用简洁的向下箭头。
- 按下控件时背景颜色加深，松开后恢复。
- “立即测试”继续支持第一次打开提醒、第二次关闭提醒。

## 六、轻提醒

- 删除圆形展开和圆形收回阶段。
- 提醒以完整矩形从屏幕顶部淡入并向下滑出。
- 关闭时矩形向上滑回并淡出。
- 顶部居中与右上角分别使用对应的屏幕顶部位置作为动画锚点。
- 每次实际显示提醒时播放一次 macOS 系统 `Glass` 提示音。
- 如果 `Glass` 不可用，调用系统默认提示音。
- 自动提醒和测试提醒使用完全相同的动画、音效与关闭逻辑。

## 七、菜单栏数字徽标

- 没有未完成任务时显示普通“待办”。
- 存在未完成任务时显示“待办”及橘黄色圆角数字徽标。
- 徽标数字使用白色，尺寸随位数适度增加。
- 徽标只强调数字，不改变整个菜单栏按钮背景。
- 使用 `NSAttributedString` 文本附件绘制徽标，保留系统状态栏按钮行为。

## 八、版本和发布

- 版本号为 v0.4.1。
- 使用现有固定本地签名、DMG 打包和 GitHub Release 流程。
- 发布说明使用中文。
- 发布前运行全部 Smoke Tests、签名检查和 DMG 校验。
- 发布后验证远端 main、tag、Release 和 DMG 指向同一提交。
- 本地删除测试构建、DMG 暂存及旧应用副本，只保留 `/Applications/MonoList.app`。

## 九、验收标准

1. Dock 点击只显示或置前清单，不会关闭已显示清单。
2. 新增待办按 Enter 后保存并自动出现下一条输入行。
3. 已有待办点击空白处能够保存。
4. 历史记录显示和隐藏具有可见的下边缘动画，窗口顶部不移动。
5. 隐藏或显示历史时，首行、已完成区域和底部间距保持固定。
6. “显示/隐藏”文字切换不产生水平位移。
7. 三个顶部按钮均为相同的圆角矩形。
8. 控制台只显示“MonoList”，两个下拉控件和测试按钮等宽。
9. 提醒直接以矩形展开和收回，并播放一次清脆系统提示音。
10. 菜单栏未完成数量显示为橘黄色数字徽标。
11. v0.4.0 能在应用内检测到 v0.4.1。
12. 发布后本地只保留一个 `/Applications/MonoList.app`。
