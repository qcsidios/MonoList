# MonoList v0.4.8 输入规则与 Release 检测修复计划

**唯一输入规则：** 只要用户正在任意待办输入框中输入，按 Enter 就保存当前内容，并立即创建、聚焦下一条空白待办。

**更新规则：** 只访问 GitHub Releases 最新页面；若发现更高版本且对应 DMG 存在，就显示升级。不得依赖 `api.github.com`。
