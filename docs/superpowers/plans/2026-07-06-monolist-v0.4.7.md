# MonoList v0.4.7 更新检测与 Enter 强化计划

**根因：**

- GitHub `api.github.com` 当前连接失败，而 `github.com/releases/latest` 与 DMG 下载地址正常；更新检测只有 API 单一路径。
- Enter 的本地事件监听在视图首次出现时捕获了焦点状态副本，后续进入草稿或编辑状态时仍可能读取旧值。

## 实施步骤

- [x] 增加 GitHub Release 跳转地址解析和下载地址生成测试。
- [x] API 检测网络失败时，回退到 GitHub 最新 Release 跳转并验证 DMG。
- [x] 增加可变输入路由状态测试，确保监听器始终读取最新焦点。
- [x] Enter 监听改为引用稳定的路由状态，不再依赖首次渲染值。
- [ ] 完整验证并发布 `v0.4.7`，本地只保留最新版 DMG。
