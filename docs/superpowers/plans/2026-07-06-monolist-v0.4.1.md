# MonoList v0.4.1 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修复连续输入、窗口动画与间距、控制台控件、提醒和菜单栏徽标，并发布可由 v0.4.0 应用内升级的 v0.4.1。

**Architecture:** 保留 AppKit 窗口承载 SwiftUI 界面的现有结构。主窗口高度使用可取消的逐帧动画器统一更新；新增待办状态继续由 `TaskDraftState` 管理；菜单栏徽标独立为轻量渲染器。提醒恢复为单阶段矩形滑入滑出，不增加新设置。

**Tech Stack:** Swift、SwiftUI、AppKit、Shell Smoke Tests、现有本地签名与 GitHub Release 流程。
