# MonoList v0.4.4 Bug 修复实施计划

> **执行要求：** 在当前独立开发分支内按任务逐项实施；所有行为改动先增加失败测试，完成后执行完整验证、清理测试产物并发布。

**目标：** 修复 Dock 与菜单栏清单开关、列表滚动弹性、多行输入高度、工具按钮对齐、控制台下拉布局、控制台表头对齐和轻提醒垂直留白。

**架构：** 菜单栏辅助进程继续负责状态项，但主动把状态项屏幕位置同步给主进程；主进程统一使用该位置处理 Dock 和菜单栏入口，并在外部点击判断中排除状态项。其余改动只调整现有 SwiftUI 布局和基于内容的尺寸计算，不改变任务、设置、签名或升级数据。

**技术栈：** Swift 6、SwiftUI、AppKit、现有 smoke test 与 GitHub Release 脚本。
