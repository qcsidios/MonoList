# MonoList v0.3 修复与紧凑界面实施计划

> **执行要求：** 使用 `executing-plans` 逐项实施；行为改动严格遵循 TDD。步骤使用复选框跟踪。

**目标：** 修复菜单栏入口、编辑、焦点、闪动和窗口布局问题，按确认稿精简控制台，并发布可保留原有数据的 `v0.3.0`。

**架构：** 将主窗口显示状态从 `NSPanel.isVisible` 的单一判断改为协调器显式管理；将草稿显示、选中和编辑状态收敛为可测试的状态模型。主清单只在自然内容高度超过上限时启用滚动；控制台删除快捷键依赖，通过回调触发提醒测试。

**技术栈：** Swift 6、SwiftUI、AppKit、现有 JSON 持久化、固定本地签名、GitHub Release。
