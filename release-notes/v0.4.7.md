# MonoList v0.4.7

本次更新强化 Enter 连续输入，并为更新检测增加 GitHub 网页回退路径。

## Enter 连续输入

- 修复键盘监听捕获首次渲染焦点状态、后续输入时仍读取旧值的问题。
- 草稿或已有待办正在输入时，按 Return 或数字键盘 Enter 都会保存并进入下一条空白待办。
- 中文输入法仍在组词时不会被提前提交。

## 更新检测

- 优先保留 GitHub API 检测。
- 当 `api.github.com` 网络连接失败时，自动回退到 GitHub 最新 Release 跳转地址。
- 回退路径会确认版本号并验证对应 DMG 可以访问，再显示升级入口。

## 兼容性

- 不改变任务、设置、签名或安装数据。
