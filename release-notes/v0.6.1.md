# MonoList v0.6.1

本次更新修正单条提醒设置的行内 UI 和浮层交互问题。

## 修复

- 待办行右侧不再为提醒按钮预留大块空白，正文区域恢复正常宽度。
- 待办行右侧只保留删除按钮；设置提醒改为通过右键菜单进入。
- 已设置提醒的状态显示在待办正文下方，例如“今天 17:30”或“每天 08:00”，不再挤占右侧操作区。
- 修复点击提醒设置浮层、今天/每天切换、小时/分钟下拉时主待办窗口被误关闭的问题。

## 优化

- 单条提醒时间选择改为“小时 + 分钟”两个下拉框。
- 小时下拉支持 00–23，分钟下拉支持 00/10/20/30/40/50。
- 下拉列表最多显示 8 项，其余内容在内部滚动。

## 可靠性

- 新增源码级 UI 回归检查，防止隐藏提醒按钮再次长期占用正文宽度。
- 新增源码级检查，防止浮层和下拉点击被误判为主窗口外点击。
