# MonoList v0.2.0

本次更新重新整理了待办的核心使用流程，并完成控制台与应用图标升级。

## 主要更新

- 应用现在同时常驻 Dock 与菜单栏，点击 Dock 图标也可以直接打开待办窗口。
- 主窗口改为“今天”单一视图，昨天未完成的任务继续保留，不显示逾期标记。
- 新任务改为列表内联输入，只有按下 Enter 才会保存；关闭窗口后仍可继续本次未提交的草稿。
- 今天完成的任务保留在列表底部；较早完成记录默认隐藏，可通过“显示 / 隐藏”展开。
- 取消独立历史记录页面，完成、恢复、删除和查看历史均在主窗口内完成。
- 已保存任务支持双击重新编辑，单击选中，点击空白处保存。
- 支持整行拖动排序；单条删除不确认，清空操作仍需二次确认。
- 主窗口会随任务数量在 148–520 pt 之间动态调整高度。
- 控制台改为固定 480 × 560 pt 的无滚动卡片布局，精简为提醒、开机启动和全局快捷键。
- 使用新的黑底白色“双态清单”Logo，并同步为 Dock 与应用图标。
- 新安装默认启用登录后自动启动，轻提醒默认 60 分钟、屏幕中间显示。

## 升级说明

- 原有任务、完成记录、设置和权限路径保持不变，升级后会继续使用原数据。
- 本版本继续采用固定本地签名，不需要 Apple 开发者认证或 App Store。
- 在应用控制台点击“检测新版本”即可下载并自动完成后续升级。
