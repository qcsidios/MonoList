# MonoList v0.6.0

本次更新新增单条待办提醒和每日重复待办能力。

## 新增

- 待办行右侧新增提醒入口，可为单条待办设置指定时间提醒。
- 单条提醒支持“今天”和“每天”两种规则。
- 时间选择采用 10 分钟颗粒度。
- 到点提醒时只展示对应的那一条待办，不影响全局轻提醒节奏。
- 每日提醒任务完成后会进入已完成；过 0 点后重新出现在未完成列表，并保留历史完成记录。
- 删除或清空未完成中的每日提醒任务，会取消对应的每日重复规则，避免第二天再次生成。

## 体验细节

- 待办行右侧使用固定操作区，提醒入口位于删除按钮左侧，避免 hover 时互相挤压。
- 已设置提醒的待办会常驻显示时间胶囊，未设置时只在 hover 或选中时显示铃铛入口。

## 可靠性

- 新增任务存储测试，覆盖一次性提醒持久化、触发后清除、每日待办跨日重新出现、同日不重复生成，以及删除/清空后不复活。
- 新增提醒调度测试，确保单条提醒不会重排全局轻提醒。
