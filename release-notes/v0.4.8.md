# MonoList v0.4.8

本次更新把 Enter 规则直接固化到所有待办输入框，并将更新检测简化为 GitHub Releases。

## Enter 规则

- 加号新增的待办：输入后按 Enter，保存并进入下一条空白待办。
- 双击空白处新增的待办：输入后按 Enter，保存并进入下一条空白待办。
- 双击已有待办进行编辑：输入后按 Enter，保存并进入下一条空白待办。
- 三种入口共用同一个文本编辑器，由编辑器直接拦截 Enter，不再依赖窗口级焦点监听。
- 中文输入法仍在选词时不会被提前提交。

## 更新检测

- 只访问 MonoList 的 GitHub Releases 最新页面。
- 从最新 Release 跳转地址读取版本号，并检查对应 DMG 是否存在。
- 不再访问或依赖 `api.github.com`。

## 兼容性

- 不改变任务、设置、签名或安装数据。
