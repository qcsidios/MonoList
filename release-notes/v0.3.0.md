# MonoList v0.3.0

本次更新重点修复菜单栏入口和待办编辑问题，并进一步压缩主清单与控制台尺寸。

## 主要更新

- 修复从其他应用点击菜单栏“待办”时偶尔无法显示清单的问题。
- 清单窗口现在贴合菜单栏展开，支持拖动，点击窗口外部即可关闭。
- 清单宽度调整为 336 pt，并只在内容超过最大高度时启用滚动。
- 修复已保存任务无法双击编辑的问题。
- 修复新增、完成和删除时整个窗口闪动的问题。
- 点击空白处会取消任务选中并结束编辑。
- 空列表继续默认显示输入行；已有任务时改为点击顶部加号后添加。
- “今天”和日期改为同一行，顶部按钮调整为加号、更多操作和齿轮。
- “已完成”入口始终保留，窗口底部不再出现无意义的大块空白。
- Dock 图标增加标准安全边距，与其他 macOS 应用图标大小一致。
- 控制台缩窄为 430 pt，并根据内容自动确定高度。
- 提醒位置精简为“顶部居中”和“右上角”。
- 增加“提醒测试”；没有任务时会显示示例提醒。
- “开机后自动启动”在新安装时默认关闭。
- 完全移除全局快捷键功能。
- 更新检测的所有过程状态和升级按钮统一显示在检测按钮右侧。

## 升级说明

- 任务、完成记录和其他设置继续使用原有数据目录，不会因升级丢失。
- 旧版“屏幕中间”或“菜单栏下方”提醒位置会自动迁移为“顶部居中”。
- 已安装用户原有的开机启动选择继续保留，不会被强制修改。
- 本版本继续使用固定本地签名和应用内自动升级，不需要 Apple 开发者认证。
