# MonoList v0.4.6

本次更新修复 Enter 连续输入和控制台下拉箭头位置。

## 待办输入

- 草稿输入和已有待办编辑统一由清单接管 Return 与数字键盘 Enter。
- 按 Enter 会保存当前待办，并立即在下一行创建新的空白待办。
- 中文输入法仍在组词时，Enter 只确认候选；组词结束后的 Enter 才保存并进入下一行。

## 控制台

- 提醒间隔和提醒位置改用 macOS 原生下拉控件。
- 文字在内容区域居中，箭头固定在灰色按钮最右侧。
- 使用标准下拉列表，整个按钮和箭头区域均可点击。

## 兼容性

- 不改变任务、设置、签名或升级数据。
