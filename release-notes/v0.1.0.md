# MonoList v0.1.0

## 更新内容

- 发布 MonoList 一栏首个可用版本。
- 支持通过菜单栏新增、编辑、完成、删除和排序待办。
- 支持已完成任务历史记录、恢复及清空。
- 支持轻提醒、开机启动和全局呼出快捷键。
- 支持每天自动检测新版本、手动检测和应用内升级。
- 使用固定本地签名保持普通升级过程中的应用身份连续性。

## 修复内容

无。

## 兼容性

- 支持 macOS 14 Sonoma 及以上版本。
- 原生支持 Apple Silicon。
- 不需要安装完整 Xcode。

## 已知限制

- 未使用 Apple Developer ID 和 Apple 公证，首次运行可能出现 Gatekeeper 提示。
- 第一版不提供账号、云同步、数据导入导出和应用内版本回退。
