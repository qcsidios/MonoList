# MonoList v0.4.3

这是一次针对菜单栏入口、清单窗口和控制台交互的稳定性修复。

## 修复

- 恢复菜单栏“待办 N”文字入口；使用独立轻量菜单栏组件，避开旧版本被 macOS 固定在不可见位置的问题。
- 清单改为无标题栏面板，默认弹出时直接贴合菜单栏下边缘。
- 修复清单窗口进入菜单栏区域的问题，并收紧任务较少时的底部留白。
- 窗口只允许从表头日期右侧的空白区域拖动，避免与任务排序冲突。
- 编辑或新增待办时，点击输入区域以外的位置会先保存内容，再取消聚焦。
- 修复提醒间隔和提醒位置下拉箭头区域无法点击的问题。

## 控制台

- “检测新版本”改为与 ShortLens 一致的紧凑灰度长条按钮。
- 检测、发现新版本、下载安装和失败状态统一显示在按钮右侧。
- “升级”按钮保持紧凑，并在安装过程中显示明确状态。
- “立即测试”文字大小与其他设置项统一。
- 控制台恢复白底和浅灰卡片样式；提醒设置的右侧箭头区域现在可以正常点击。

## 数据与升级

- 本次升级不会清除任务、历史记录或设置。
- 主应用 Bundle ID 与固定本地签名保持不变，避免升级造成权限、用户数据和应用身份变化。
