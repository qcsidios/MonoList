# MonoList v0.4.3

这是一次针对菜单栏入口、清单窗口和控制台交互的稳定性修复。

## 修复

- 恢复原生菜单栏“待办 N”入口，移除会干扰菜单栏状态项的自定义包装。
- 清单改为无标题栏面板，默认弹出时直接贴合菜单栏下边缘。
- 窗口只允许从表头日期右侧的空白区域拖动，避免与任务排序冲突。
- 编辑或新增待办时，点击输入区域以外的位置会先保存内容，再取消聚焦。
- 修复提醒间隔和提醒位置下拉箭头区域无法点击的问题。

## 控制台

- “检测新版本”改为与 ShortLens 一致的紧凑灰度长条按钮。
- 检测、发现新版本、下载安装和失败状态统一显示在按钮右侧。
- “升级”按钮保持紧凑，并在安装过程中显示明确状态。
- “立即测试”文字大小与其他设置项统一。

## 数据与升级

- 本次升级不会清除任务、历史记录或设置。
- Bundle ID 与固定本地签名保持不变，避免升级造成权限和身份变化。
