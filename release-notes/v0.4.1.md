# MonoList v0.4.1

本次更新集中改善连续录入、窗口稳定性和轻提醒体验。

## 新增

- 新增待办按 Enter 保存后自动进入下一行，连续录入更顺手。
- 菜单栏增加橘色未完成数量徽标。
- 控制台增加“提醒声音”开关，默认开启。

## 优化

- Dock 点击改为显示或置前清单，菜单栏入口继续支持显示与隐藏。
- 历史记录和任务变化时，窗口从下边缘平滑展开或收起。
- 固定列表顶部、已完成区域和窗口底部间距。
- 统一右上角三个操作按钮的圆角矩形样式。
- 控制台应用名称简化为“MonoList”。
- 提醒间隔、提醒位置和立即测试控件统一宽度与字体。
- 轻提醒改为顶部矩形滑入滑出，并使用 macOS 系统清脆提示音。

## 数据与升级

- 升级不会清除任务、历史记录或设置。
- 继续使用固定本地签名，保留现有权限和应用数据。
