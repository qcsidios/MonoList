# MonoList v0.5.0

本次更新升级轻提醒控制台，并优化待办拖拽排序体验。

## 轻提醒控制台

- 新增“提醒时段”设置，可选择每天允许提醒的起止时间。
- 默认提醒时段为 `09:00 - 22:00`，旧设置文件会自动使用该默认值。
- 新增“下次提醒”展示框：
  - 已启用且有待办时显示预计提醒时间。
  - 未启用时显示“未启用”。
  - 没有待办时显示“暂无待办”。
- 如果下一次提醒会落在提醒时段之外，会自动顺延到下一个允许时段。

## 拖拽排序

- 待办拖拽调整顺序时加入柔和移动动画，更接近苹果提醒事项的动态反馈。
- 保留 v0.4.11 的透明拖拽预览，避免拖动时出现重影。

## 兼容性

- 不改变任务数据、签名或安装数据。
- 设置文件向后兼容；旧版本缺失的新提醒时段字段会自动补默认值。
